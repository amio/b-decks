## Two User Faced Product

Cloud IDE x Web IDE

---

- Cloud IDE
  - Web Editor + __Remote Runtime__
  - 资源成本高，有网络延迟，功能强大
  - 场景：持续研发、调试、发布的重度研发场景
- Web IDE
  - Web Editor + __Web Container__
  - 零边际成本，轻便快捷
  - 场景：即改即走的轻量级编辑场景
